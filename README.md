I design and build production-grade systems at the intersection of **AI/ML**, **fraud risk**, and **regulated financial infrastructure**. My work spans the full spectrum from real time transaction monitoring engines to intelligent content automation pipelines.

My engineering philosophy is anchored in **DIRFT**  *Do It Right First Time*. Structured logic, clean architecture, and deliberate design choices over endless iteration on fragile foundations.
